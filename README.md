## 🌟 Features That Shouldn't Exist (But Do!)

### 🎹 Animal Sound Orchestra
- 12 different animal instruments with unique sound patterns
- Each animal has its own weird personality and sound effects
- Professional-level weirdness with multiple oscillators and effects

### 🎤 Voice Transformation Studio
- Record your voice and transform it into:
  - 🤖 Robot Mode (beep boop beep)
  - 🐿️ Chipmunk Style (squeaky supreme)
  - 👹 Demon Voice (spooky scary)
  - 🌊 Underwater (glub glub)
  - 👽 Alien Speech (extraterrestrial excellence)
  - 🔄 Backwards (sdrawkcab)

### 🎲 Random Generators
- **Song Generator**: Creates completely random song structures with titles like "Mysterious Taco Dancing"
- **Sound Mixer**: Combines 5 random animal sounds simultaneously
- **Circus Mode**: Full disco experience with floating emojis

### 🌪️ Chaos Mode
- Automatically plays random sounds every second
- Screen-shaking effects
- Maximum weirdness overload
- Use spacebar to toggle chaos!

### 🏆 Achievement System
- Unlock hilarious achievements like "Chaos Master" and "Professional Noise Maker"
- Track your journey from normal human to animal sound virtuoso

### ⌨️ Keyboard Controls
- `A-L` keys: Play different animal sounds
- `Spacebar`: Toggle Chaos Mode
- Click anywhere for maximum interaction!

## 🚀 Quick Start (Prepare for Weirdness)

1. **Clone this masterpiece:**
   ```bash
   git clone [your-repo-url]
   cd weird-music-maker
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Launch the chaos:**
   ```bash
   npm start
   # OR
   npm run dev
   ```

4. **Open your browser to `http://localhost:3000`**

5. **Click everything and make noise!**
